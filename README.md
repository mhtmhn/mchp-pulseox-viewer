## Get started

Install the dependencies:

Install latest [Node.js](https://nodejs.org/en) (v22.10.0 or later)

Enable corepack on an **admin** console:

```bash
corepack enable
```

Run in root of repository:

```bash
yarn
```

Start coding:

```bash
yarn dev
```

## Building

```bash
yarn build
```

Test built project:

```bash
yarn serve
```
